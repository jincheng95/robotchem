

Settings files
==============

local_settings.py
-----------------

Contains one constant, ``ACCESS_CODE``, which stores the access code to 'authenticate' communication with the server. This setting is separated from the main ``settings.py`` configuration file to prevent Git pushing this value to GitHub.
